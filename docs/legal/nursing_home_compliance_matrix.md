# 요양병원 법적 규제 매트릭스 (ThinQ Workspace Sentinel)

> 9개 법령·고시·평가 기준 매핑. 모든 자동 증빙은 `sentinel.legal_mappings` 테이블에 영구 저장.

## 1. 의료법 제36조 — 감염관리위원회 의무

| 항목 | 내용 |
|---|---|
| **소관** | 보건복지부 |
| **의무** | 200병상 이상 병원·요양병원은 감염관리위원회 + 감염관리실 설치 의무 |
| **하위 규정** | 「의료기관의 감염관리 기준」 (보건복지부 고시 제2022-XXX호) |
| **Sentinel 증빙** | `alerts` + `rehva_results` + `smart_protocol_log` → 월간 PDF 자동 생성 |
| **위반 시** | 시정명령·과태료 (의료법 제92조) |

## 2. 감염병예방법 제16조 — 감염병 발생 신고

| 항목 | 내용 |
|---|---|
| **소관** | 질병관리청 |
| **의무** | 1군·2군 감염병 24h 내 보건소 신고 |
| **대상 (요양병원 핵심)** | COVID-19·결핵·인플루엔자·노로 |
| **Sentinel 증빙** | Tier `ALERT` 이상 진입 시 자동 보건소 알림 + 신고 양식 PDF |
| **위반 시** | 200만원 이하 과태료 |

## 3. 산업안전보건법 제5조 — 근로자 보건 조치

| 항목 | 내용 |
|---|---|
| **소관** | 고용노동부 |
| **의무** | 요양보호사·간호조무사 작업환경 보건 조치 |
| **하위 규정** | 「산업안전보건기준에 관한 규칙」 — 환기·CO·CO2·온습도 |
| **Sentinel 증빙** | `sensor_readings` 1년 보관·연간 작업환경측정 보고서 |
| **위반 시** | 5년 이하 징역 또는 5천만원 이하 벌금 |

## 4. 실내공기질관리법 제5조

| 항목 | 내용 |
|---|---|
| **소관** | 환경부 |
| **대상** | 「다중이용시설」 — 의료기관·요양시설 포함 |
| **유지 기준** | PM10 ≤75·PM2.5 ≤35·CO2 ≤1000·HCHO ≤80·총부유세균 ≤800 |

| **위반 시** | 개선명령 → 1천만원 이하 과태료 |

## 8. 요양병원 적정성평가 — 감염관리 영역

| 항목 | 내용 |
|---|---|
| **소관** | 건강보험심사평가원 (HIRA) |
| **주기** | 연 1회 |

| **재무 효과** | 1등급 유지 시 200병상 연 +5~8억 증수 |

## 개인정보·보안 (ISMS-P 영향도)

| 항목 | 처리 방법 |
|---|---|
| **입소자 식별정보** | `users.anonymized_id` (SHA-256 해시), 원본 미저장 |
| **재실 인원 데이터** | 집계값(count)만 저장, 개인 동선 미저장 |
| **센서 원본** | TimescaleDB Hypertable + 1년 retention policy (이미 적용) |
| **PHI (Protected Health Info)** | 시스템 외부 미전송, Slack 알림은 비식별 메시지만 |
| **접근 통제** | RBAC 6 roles (ICN·DIRECTOR·FM·CAREGIVER·RESIDENT·GUARDIAN) |
| **감사 로그** | `audit_log` 테이블 (다음 마이그레이션 003에서 추가) |
