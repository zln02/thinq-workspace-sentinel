# ROI — 감염예방관리료 등급 차등 (B2G 지불근거 슬라이드)

> 목적: "선제 감염대응"이라는 증명 어려운 효능 대신, **즉시 지불 가능한 수가(돈) 근거**로 사업성을 못박는다.
> ⚠️ 아래 [확정필요] 표시 금액은 발표 전 **건강보험심사평가원(심평원) 최신 수가표**로 채워야 함(연도별 변동).

---

## 1. 왜 지금이 타이밍인가 (시장 형성기)

- **감염예방관리료 등급별 급여 차등** 시행 (심평원, 2024.10~). 감염관리 활동·인력·실적이 등급을 가르고, 등급이 **급여(수가)** 에 직결.
- **의료관련감염 예비평가** 2025년부터 본격 진입 → 요양병원이 감염관리 "증빙·문서화" 부담을 안게 됨.
- 즉, **감염관리 자동 증빙 = 등급 유지/상향 = 수가 = 돈**. 시장이 막 열리는 구간.

## 2. ROI 산식 (요양병원 단위)

```
연간 가치 V = (등급 상향/유지로 확보한 수가 차액)          ← 심평원 수가표 [확정필요]
            + (감염관리 행정·문서화 인건비 절감)           ← 간호인력 X시간/주 × 시급
            + (집단감염 1건 예방 시 기대손실 회피)          ← 폐쇄·보상·평판 (보수적 계상)

도입비용 C = SW 구독료(연) + 센서노드 HW(공간당 1회)

회수기간 = C / V   →  목표: 12개월 이내
```

### 입력 템플릿 (150병상 가정)

| 항목 | 값 | 출처/비고 |
|---|---|---|
| 등급 차등 수가 차액(연) | [확정필요] 원 | 심평원 감염예방관리료 수가표 |
| 감염관리 문서화 절감 시간 | 주 X 시간 | 간호인력 인터뷰/추정 |
| 간호인력 시급 | 약 [확정필요] 원 | 보건의료 임금통계 |
| 집단감염 회피 기대값 | 보수적 1건/3년 환산 | 폐쇄손실+행정처분 |
| SW 구독료(연) | 가격모델 3-tier 연동 | `docs/business/pricing-model.md` |
| 센서노드 HW | 공간당 약 5~7만원 | SHT40+SCD41+카메라(노트북 공유) |

## 3. 발표 한 줄

> "저희가 파는 건 '감염 예방'이라는 증명 어려운 효능이 아니라, **감염관리 활동의 자동 증빙입니다.**
> 2024년 10월부터 감염예방관리료가 등급제로 바뀌면서, 감염관리 점수가 곧 수가가 됐습니다.
> 저희 시스템은 환기 대응 이력을 자동 기록해 그 점수를 채우고, 도입비는 12개월 내 회수됩니다."

## 4. QA 방어

- **Q. 수가 차액이 실제로 그만큼 되나?** → [확정필요] 심평원 수가표 인용. "등급 1단계 차이 = 연 N원" 명시.
- **Q. 너희가 점수를 보장하나?** → "점수 산정은 심평원 권한. 우리는 **증빙 자동화로 충족 가능성을 높이고 행정부담을 없앤다**"로 한정(효능 단정 회피).
