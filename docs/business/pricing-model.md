# ThinQ Workspace Sentinel — 가격 모델 (3-tier)

> B2G 요양병원 감염관리 SaaS + 가전 자동방역. 구독 / 건당 / PoC 3-tier.
> 실제 금액은 심평원 수가·LG 가전 단가·운영비 확정 후 채운다([확정필요]).
> 관련: [ROI_감염예방관리료](../발표/ROI_감염예방관리료.md)

---

## 1. 가격 3-tier

| Tier | 대상 | 과금 방식 | 가격(월/병상 기준) | 포함 |
|---|---|---|---|---|
| **PoC** | 신규 도입 검증 병원 | 3개월 고정 + 무상 컨설팅 | **[확정필요]** (예: 300만원/3개월·정액) | 센서킷 1세트·대시보드·리포트, 효과 측정 |
| **구독(Subscription)** | 상시 운영 요양병원 | 병상수 × 월정액 | **[확정필요]** (예: 8,000~12,000원/병상·월) | 전 페르소나 대시보드·외부 조기경보 연동·가전 자동제어·적정성평가 증빙 자동화·업데이트 |
| **건당(Pay-per-use)** | 다지점·간헐 운영 | 자동대응 이벤트 건당 | **[확정필요]** (예: 자동방역 1건당 N원) | 인프라 공유, 사용량 기반 |

> 200병상 구독 예시: 200 × [월정액] = **[확정필요]/월** → 연 [확정필요]. 적정성평가 1등급 유지 시 수가 가산(연 5~8억 추정, ROI 문서 §2)으로 회수.

---

## 2. 가격 책정 근거 (cost-plus + value-based)

- **원가(COGS)**: 센서킷(RPi+아두이노+DHT·PM·CO2 [확정필요]원), 카메라(노트북 활용 시 0), 가전 연동(고객 보유 ThinQ/SmartThings 활용), 클라우드(GCP [확정필요]원/월·병원), 유지보수 인건.
- **가치 기반**: 핵심 가치는 *비용 절감*이 아니라 **적정성평가 등급 유지 → 수가 가산** + **집단감염 1건 회피(병상 폐쇄·소송·평판 손실 회피)**. 가격은 회피 손실의 일부를 환수하는 수준으로 책정.
- **경쟁 비교**: 단순 IAQ 모니터링 SaaS([확정필요]원/월) 대비 *외부 역학 선제 + 가전 자동제어 + 법규 증빙 자동화*로 프리미엄 정당화.

## 3. 기능 변경 시 가격 영향 (변경관리)

기능 추가/제거 시 본 3-tier 모델에 반영 여부를 PR 설명에 명시한다(전역 규칙).
- 가전 제어 종류 추가(8종 Smart Protocol) → 구독 상위 플랜 차등.
- 보호자 PWA → 별도 옵션(병상당 [확정필요]원) 또는 구독 포함.
- 생성형 AI 브리핑/리포트(LLM) → API 비용 전가 옵션.
