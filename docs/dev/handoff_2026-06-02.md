## [Critical] 4가지 불일치

### 1. ML 스택 — SHAP/GAM 사용 금지

- **설계서:** 앙상블 XGB+ARIMA+GAM+SHAP
- **코드 실재:** XGBoost (주모델) + Granger 인과검정 + TFT (PoC 합성데이터)
- **시연 화면 가이드:**
  - ❌ SHAP force plot · GAM contribution plot 절대 만들지 마
  - ✅ XGBoost `feature_importances_` (gain) bar plot
  - ✅ Granger causality p-value 표 (4 lag × 5 변수)
  - ✅ TFT attention heatmap (PoC 합성 / 실데이터 전환 로드맵)
- **발표 대본:** "XGBoost + Granger" 가 정답. PPT p17 그대로 사용.

### 2. CO2 법적 기준 — 1500 → 1000ppm

- **설계서 FR-14:** 1500ppm  ← 오류
- **법적 기준:** 실내공기질관리법 의료기관 = **1000ppm**
- **코드 시드값:** 1000 (일치)
- **시연 화면 가이드:**
  - CO2 게이지 임계선 = 1000ppm
  - "심사위원 입김 → 600 → 1000ppm 격상" Mode 2 라이브 데모
  - 발표 시: "실내공기질관리법 의료기관 기준입니다"
