## 핵심 원칙

### 색맹 안전 상태색 (Wong 2011 + Carbon DS)
색만으로 단계 구분 금지(남성 8%·고령 15% 색각이상). 색+아이콘+굵기+텍스트 4중.
```
MONITOR 정상  ●✓  #009E73(청록)   CAUTION 주의 ◆⚠ #E69F00(앰버)
ALERT 경보   ▲🔔 #D55E00(버밀리온) HIGH_RISK 고위험 ■!! #CC79A7(보라핑크)
CRITICAL 위급 점멸💢 #0072B2(파랑, Carbon 방식)
```
**브랜드 LG레드(#A50034)는 상태색 금지** — 헤더/로고/버튼 전용 격리.

### 가독성 (PMC7330495)
- KPI 수치 32px bold, 위험레이블 20px bold, 본문 14px
- 대비 WCAG AA 4.5:1 — 흰 배경 위 앰버는 #B45309로 진하게
- 숫자: `1,420 ppm (기준 1,000)` 천단위+정상범위 병기

### 실시간 UX (Smashing 2025)
- 신선도: ●방금/○23초전/△2분전(앰버)/✕없음(적). 테두리색에도 반영
- 스파크라인 60px 추가(추세 즉시 파악)
- **비CRITICAL 움직임 금지**(animate-ping/bounce) — attention fatigue. CRITICAL만 1.5초 pulse

### 알람 운영
- acknowledge 플로우(경보↑는 "확인" 눌러야 사라짐)
- "즉각조치/모니터링" 탭 분리

### B2G 신뢰 요소
- 타임스탬프 3층(시스템/센서수집/로그 ISO8601)
- 데이터 출처 배지([실측]vs[시뮬]) — 감사추적 핵심
- 기관 3단 병기(병원명+보건소+LG ThinQ), 시스템상태 푸터

### 보호자앱 (안심 중심)
- 기본=안심, 이상 시만 경고. 정보 3계층(상태어→바이탈→raw)
- 폰트 18px+(보호자도 60~70대), ALERT+만 푸시

## 적용 매핑
- 타임스탬프/출처배지 → dashboard.html KPI 카드, FloorPlan RoomCard
- 색상 → globals.css / dashboard.html CSS 변수 분리
- 보호자 → family/page.tsx 조건부 배경(현 animate-pulse 상시 → tier 조건부)
