# Prior Art 비교 & 특허 방어 (차별점 증명 슬라이드)

> 목적: "우리가 새롭다"를 주장이 아닌 **실제 선행기술 옆에 놓고 비교**해 증명한다.
> ⚠️ 발표 전 특허번호·제품 URL **재확인** 필수(검색 기반, 청구범위 원문 대조 권장).

## 1. 가장 가까운 선행기술 3종

| 구분 | 무엇 | 한계(=우리 빈틈) |
|---|---|---|
| 상용제품 | **75F "Epidemic Mode"** (빌딩 BMS) | CDC/ASHRAE 가이드로 외기댐퍼 최대개방·Smart Purge. **단, 외부 역학 API 실시간 자동연동 없음** — 관리자가 수동으로 파라미터 변경 |
| 특허 | **US12264828** (지역 유행 균주 → HVAC·살균 제어) | **상업용 빌딩 / 정적 균주정보** 대상. 공공 API 실시간 피드포워드·요양병원 소형가전·선행지표(검색량) 아님 |
| 학술 | Wells-Riley + IoT 환기제어 논문군 | **실내 센서 피드백 루프에 한정**. 외부 역학 신호를 제어 입력으로 통합한 사례 없음 |

## 2. 비교표 (발표용 핵심)

| 기능 | LG ThinQ/퓨리케어 | 코웨이·삼성 | 75F Epidemic | US12264828 | **우리** |

## 3. 특허 방어 문장 (US12264828 충돌 회피)

> "유사 특허가 있으나 그 청구범위는 **상업용 빌딩의 정적 균주정보 기반 HVAC 제어**입니다.
> 본 시스템은 ① **질병청·네이버 DataLab 공공 API의 실시간 피드포워드**, ② **요양병원 소형 가전(ThinQ) 단위 제어**,
> ③ **공식 집계 1~2주 선행하는 검색 트렌드 선행지표** 라는 점에서 적용영역·신호원·시간성이 구분됩니다.
> 향후 출원 시 이 세 축을 청구 차별점으로 둡니다."

## 4. 차별화 핵심 한 줄

> "외부 역학 신호를 가전 제어 루프의 **피드포워드(feedforward) 입력**으로 통합한 상용·요양병원 솔루션은 현재 없습니다.
> 기존은 전부 실내 센서 **피드백(feedback)** 에 갇혀 있습니다."
